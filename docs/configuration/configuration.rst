Configuration
=============

Deploy the ZooKeeper and Kafka and update the following 2 properties in MsgRtrApi.properties

config.zk.servers=<<zookeeper_host>>\n
kafka.metadata.broker.list=<<kafka_host>:<<kafka_port>>\n
